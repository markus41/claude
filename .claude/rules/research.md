# Research Rules

- ALWAYS use MCP tools for web research instead of WebFetch/WebSearch:
  - Perplexity MCP (`mcp__perplexity__*`) for knowledge queries and current information
  - Firecrawl MCP (`mcp__firecrawl__*`) for scraping specific web pages
  - Context7 MCP (`mcp__plugin_context7_context7__*`) for library documentation
- Use subagents for research to keep main context clean
- Document research findings in relevant files, not just conversation
- When researching errors, check `.claude/rules/lessons-learned.md` first
