# lobbi-m365-automator Context Summary

## Plugin purpose
Microsoft 365 automation for insurance and financial services firms. Covers SharePoint site and library provisioning, Teams channel/tab setup, Power Automate flow generation, Exchange mail rules, tenant-level M365 auditing, and OneDrive structure organization.

## Skill index
- `skills/sharepoint-builder/SKILL.md` — SharePoint site, list, library, and page provisioning specs
- `skills/teams-provisioner/SKILL.md` — Teams team, channel, tab, and app provisioning specs
- `skills/power-automate-gen/SKILL.md` — Power Automate flow design for business process automation
- `skills/exchange-rules/SKILL.md` — Exchange Online mail flow rules and shared mailbox configuration
- `skills/m365-auditor/SKILL.md` — M365 audit log queries and compliance reporting specs
- `skills/onedrive-organizer/SKILL.md` — OneDrive folder structure and governance policy design

## Agent index
- `agents/m365-architect.md` — Architects complete M365 workspace solutions for clients
- `agents/power-platform-specialist.md` — Designs Power Automate flows and Power Apps integrations
- `agents/m365-compliance-advisor.md` — Advises on M365 compliance, retention, and governance settings

## When-to-load guidance
Load this summary first. Open specific skill files based on the M365 service being automated.
