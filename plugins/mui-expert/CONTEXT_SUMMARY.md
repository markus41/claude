# MUI Expert Plugin

Comprehensive Material UI expert for React applications. Covers 25 skill areas:
theming, CSS variables, Pigment CSS, components, sx/styled API, slots API,
MUI X (DataGrid, DatePickers, Charts, TreeView), accessibility, performance,
SSR/Next.js, animations, virtualization, forms, white-label/multi-tenant,
headless components (MUI Base), Joy UI, i18n/RTL, testing, and migration.

## Commands

| Command | Purpose |
|---------|---------|
| `/mui-theme` | Generate or audit a MUI theme |
| `/mui-component` | Scaffold a MUI component with best practices |
| `/mui-audit` | Audit MUI usage for performance, a11y, and anti-patterns |
| `/mui-migrate` | Migrate MUI code between versions |
| `/mui-datagrid` | Build or configure MUI X DataGrid |
| `/mui-design-system` | Scaffold a complete design system with tokens |
| `/mui-charts` | Scaffold MUI X Charts with composition API |
| `/mui-perf` | Analyze and optimize MUI performance |
| `/mui-ssr` | Configure MUI for Next.js SSR |
| `/mui-white-label` | Generate multi-tenant white-label theme architecture |
| `/mui-slots` | Customize component internals via slots API |
| `/mui-setup` | Install packages, configure MCP server, LSP, theme, and framework |

## MUI MCP Server

The official `@mui/mcp` server is configured in `.mcp.json` as `mui-mcp`.
Use `useMuiDocs` and `fetchDocs` tools to query MUI's official documentation
for accurate, up-to-date API details, migration guidance, and complex questions.

## Agents

| Agent | Role |
|-------|------|

| `mui-a11y-auditor` | WCAG compliance audit with auto-fix |

| `mui-migration-specialist` | Version migration analysis and execution |
