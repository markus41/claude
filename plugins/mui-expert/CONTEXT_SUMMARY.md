# MUI Expert Plugin

Material UI expert for React applications. Covers theming, component implementation,
sx/styled API, MUI X (DataGrid, DatePickers), accessibility, performance, and migration.

## Quick routing

| Need | Resource |
|------|----------|
| Create/customize theme | `skills/theming/SKILL.md` |
| sx prop or styled() API | `skills/sx-styled/SKILL.md` |
| Component implementation | `skills/components/SKILL.md` |
| DataGrid (MUI X) | `skills/data-grid/SKILL.md` |
| DatePicker (MUI X) | `skills/date-pickers/SKILL.md` |
| Forms & validation | `skills/forms-validation/SKILL.md` |
| Layout & responsive | `skills/layout-responsive/SKILL.md` |
| Accessibility (a11y) | `skills/accessibility/SKILL.md` |
| Bundle size / perf | `skills/performance/SKILL.md` |
| v5 → v6 migration | `skills/migration/SKILL.md` |

## Commands

| Command | Purpose |
|---------|---------|
| `/mui-theme` | Generate or audit a MUI theme |
| `/mui-component` | Scaffold a MUI component with best practices |
| `/mui-audit` | Audit MUI usage for performance, a11y, and anti-patterns |
| `/mui-migrate` | Migrate MUI code between versions |
| `/mui-datagrid` | Build or configure MUI X DataGrid |

## Agents

| Agent | Role |
|-------|------|
| `mui-architect` | Theme system design, component library planning |
| `mui-reviewer` | Code review for MUI best practices |
| `mui-migration-specialist` | Version migration analysis and execution |
