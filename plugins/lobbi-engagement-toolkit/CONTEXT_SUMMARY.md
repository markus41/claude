# lobbi-engagement-toolkit Context Summary

## Plugin purpose
Client engagement and project delivery tools for The Lobbi's fixed-scope, fixed-price automation model. Covers pre-sale (ROI analysis, scope definition, proposals) through delivery (change order protection, milestone tracking, post-launch review).

## Skill index
- `skills/scope-define/SKILL.md` — Generate project scope documents with explicit inclusions/exclusions
- `skills/proposal-gen/SKILL.md` — Create client proposals with ROI projections and timeline
- `skills/roi-calculator/SKILL.md` — Calculate automation ROI from current manual effort data
- `skills/change-order-guard/SKILL.md` — Detect scope creep and generate change order requests
- `skills/delivery-tracker/SKILL.md` — Generate milestone tracking and progress reporting configs
- `skills/post-launch-review/SKILL.md` — Post-go-live assessment and optimization checklist

## Agent index
- `agents/project-manager.md` — Manages delivery milestones and client communication
- `agents/roi-analyst.md` — Calculates and presents automation ROI
- `agents/scope-architect.md` — Defines project scope boundaries and acceptance criteria

## When-to-load guidance
Load this summary first. Open specific skill files based on the engagement phase (pre-sale vs. delivery).
