# Skills Index — draw.io Diagramming Plugin

Quick reference for which skill to load based on task.

## Core Skills

| Skill | Path | Load When |
|-------|------|-----------|
| XML Generation | `xml-generation/SKILL.md` | Building any draw.io XML (structure, styles, shapes, edges) |
| Templates | `templates/SKILL.md` | Need a starting template for a diagram type |
| Quality Critique | `quality-critique/SKILL.md` | Reviewing/validating generated diagram before writing |
| Diagram Types | `diagram-types/SKILL.md` | Choosing the right diagram type for a use case |
| AI Generation | `ai-generation/SKILL.md` | AI-powered diagram creation patterns |

## Research & Enrichment

## Load Order for Common Tasks

**Creating a new diagram:**
1. `templates/SKILL.md` — get the skeleton
2. `xml-generation/SKILL.md` — reference for styles/shapes
3. `quality-critique/SKILL.md` — validate before writing

**Enriching an existing diagram:**
1. `research-agents/SKILL.md` — spawn researchers
2. `xml-generation/SKILL.md` — reference for modifications

**Embedding a diagram:**
1. `platform-integrations/SKILL.md` — platform-specific guidance
