# lobbi-bi-reports Context Summary

## Plugin purpose
Business intelligence and reporting for insurance and financial services executives. Covers Power BI report and dashboard design, KPI definition frameworks, embedded analytics specs, data model mapping, automated refresh scheduling, and report distribution workflows.

## Skill index
- `skills/power-bi-builder/SKILL.md` — Power BI report and visual design specifications
- `skills/kpi-designer/SKILL.md` — KPI definition framework with targets, thresholds, and owners
- `skills/dashboard-embedder/SKILL.md` — Embedded analytics integration specs for portals and apps
- `skills/data-model-mapper/SKILL.md` — Power BI data model design and relationship mapping
- `skills/refresh-scheduler/SKILL.md` — Dataset refresh scheduling and failure alerting configs
- `skills/report-distributor/SKILL.md` — Automated report distribution via email and Teams

## Agent index
- `agents/bi-architect.md` — Architects end-to-end BI solutions from data source to executive dashboard
- `agents/kpi-analyst.md` — Defines and validates KPI frameworks with business stakeholders
- `agents/dashboard-designer.md` — Designs Power BI dashboards for usability and insight clarity

## When-to-load guidance
Load this summary first. Open specific skill files based on the BI component being designed or built.
