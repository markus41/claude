# lobbi-data-migration Context Summary

## Plugin purpose
Data migration planning and execution for insurance and financial services system transitions. Covers source schema analysis, data quality profiling, migration strategy planning, ETL transform design, pre/post-migration validation, and rollback coordination.

## Skill index
- `skills/schema-analyzer/SKILL.md` — Analyze source and target schemas and map structural differences
- `skills/data-profiler/SKILL.md` — Profile data quality, completeness, and anomalies before migration
- `skills/migration-planner/SKILL.md` — Build phased migration plan with cutover strategy and risk register
- `skills/transform-builder/SKILL.md` — Design ETL transformation rules for field mapping and data cleansing
- `skills/validation-runner/SKILL.md` — Pre and post-migration validation checklist and reconciliation specs
- `skills/rollback-coordinator/SKILL.md` — Rollback plan design with decision criteria and execution steps

## Agent index
- `agents/migration-architect.md` — Architects end-to-end data migration strategy and execution plan
- `agents/data-quality-analyst.md` — Profiles source data and identifies cleansing requirements
- `agents/rollback-specialist.md` — Designs rollback procedures and validates recovery readiness

## When-to-load guidance
Load this summary first. Open specific skill files based on the migration phase being planned or executed.
