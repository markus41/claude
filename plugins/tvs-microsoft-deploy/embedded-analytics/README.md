# Embedded Analytics Accelerator

This package provides a reusable foundation for delivering multi-tenant embedded analytics with Microsoft Fabric + Power BI and Power Pages.

## Contents

- `reference-architecture/`: design guidance for tenant isolation, token issuance, semantic model strategy, and row-level security.
- `templates/`: deployment-ready starter templates for Power Pages embedding and Fabric/Power BI semantic model rollout.
- `onboarding/`: client onboarding checklist for security, legal, identity, and operational readiness.
- `docs/`: sales-facing collateral for packaging, implementation timelines, and support model.

## Intended Outcomes

- Consistent per-client workspace provisioning.
- Predictable semantic model tenancy patterns based on client size and compliance.
- Secure embed token strategy for browser delivery.
- Practical implementation plan and support coverage guidance for GTM teams.
