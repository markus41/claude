# Implementation Timeline

## Week 0: Discovery & Contracting

- Confirm business outcomes, personas, and target KPIs.
- Complete onboarding checklist (identity, legal, security, SLA).

## Week 1-2: Foundation

- Provision workspace(s), capacities, and service principals.
- Configure Power Pages site and secure embed API.
- Establish CI/CD baseline and environment promotion path.

## Week 3-4: Data & Semantic Layer

- Map source data to conformed model.
- Deploy semantic model and RLS roles.
- Validate refresh schedules and data quality thresholds.

## Week 5: UX & UAT

- Embed reports in Power Pages.
- Execute UAT scripts, access tests, and performance tuning.

## Week 6: Go-Live & Hypercare

- Production cutover with monitored rollout.
- Hypercare support, defect triage, and adoption handoff.
