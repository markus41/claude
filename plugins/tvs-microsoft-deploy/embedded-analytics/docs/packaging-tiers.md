# Packaging Tiers (Sales-Ready)

## Starter

- Shared deployment baseline with secure embedding.
- Standard KPI/report package.
- Monthly release train.
- Best for first-time embedded analytics launches.

## Growth

- Dedicated client workspace and enhanced governance.
- Expanded metrics catalog and regional deployment choices.
- Bi-weekly release cycle, prioritized backlog intake.
- Ideal for scaling clients with moderate customization needs.

## Enterprise

- Dedicated workspace + dedicated semantic model option.
- Advanced security controls, contracted residency, and audit workflows.
- Custom roadmap governance with architecture board cadence.
- Suitable for regulated and high-scale programs.
