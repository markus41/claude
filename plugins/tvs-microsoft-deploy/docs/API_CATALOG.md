# TAIA Integration API Catalog

## Microsoft APIs

### Microsoft Graph
- Planner: plans, buckets, tasks
- Teams: channels, messages, memberships
- SharePoint/OneDrive: files, permissions
- Users/Groups: lifecycle and access control
- Excel Workbooks: range operations, table updates

### Fabric REST API
- Workspaces, Lakehouses, Notebooks
- Pipeline and job orchestration
- Capacity and workload monitoring

### Power Platform APIs/CLI
- Dataverse metadata/data operations
- Solution import/export and ALM transport
- Power Pages deployment integration

### Azure APIs
- ARM/Bicep deployment operations
- Key Vault secret/config management
- App Insights diagnostics and KQL queries

## Non-Microsoft APIs
- Stripe
- Firebase
- Slack
- DocuSign
- HubSpot

## Embedded Analytics Selling Pack
- Multi-tenant embed architecture with RLS.
- Power Pages embedding model for client portals.
- Token service model with short-lived embed tokens and audit logging.
- Pricing tiers: starter (single report), growth (domain bundle), enterprise (custom semantic model).
